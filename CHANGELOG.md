# CHANGELOG (版本更新紀錄)

本專案的所有版本更新皆會記錄於此，並同步於 Git 標籤中。

---

## [v1.0.4] - 2026-06-24

### 🚀 新增與優化
* **新增手動更新腳本 (`update_manual.py`)**：當官網防爬蟲 WAF (Cloudflare/Incapsula) 阻擋自動抓取時，使用者可透過簡單的一行 CLI 指令手動輸入並更新所有情緒指標數值，系統會自動重新計算分數與亮燈狀態。
* **新增 `.gitignore`**：過濾專案中的 python 快取與系統暫存檔案。

---

## [v1.0.3] - 2026-06-22

### 🚀 新增與優化
* **自訂燈號分數門檻**：調整 `fetch_data.py` 中的 `light_status` 判定邏輯，以符合使用者的情緒範例模板：
  * 總分累計大於等於 **3 分亮紅燈**（原為 4 分）。
  * 總分累計大於等於 **2 分亮黃燈**（原為 1 分）。
  * 總分累計小於 2 分維持綠燈。

---

## [v1.0.2] - 2026-06-22

### 🚀 新增與優化
* **NAAIM 區段色彩**：將 NAAIM 經理人持倉進度條優化為與截圖相符的三色漸層（0-30 為紅、30-55 為黃、55-100 為灰藍/綠），並加粗加高分界線。
* **數據防消失備份機制 (Fallback)**：在 `fetch_data.py` 中實作備份讀取機制。若因 WAF 或網路限制導致 AAII 或 NAAIM 抓取失敗時，會自動保留上次成功的歷史數值，不會清空網頁上已有的指標數據。

### ⚙️ 調整與修正
* **AAII/NAAIM 數值校正**：手動校正與補回 AAII 的看漲 (36.6%)、中性 (24.1%)、看跌 (39.4%) 與 NAAIM (92.83) 的真實數值，使首頁渲染與使用者提供之最新截圖一致。

---

## [v1.0.1] - 2026-06-22

### 🚀 新增與優化
* **指標視覺化卡片**：將原本的文字清單改版為 4 個精美的儀表板卡片，支援：
  * **VIX**：動態級距條、黑圓點指針、波動狀態徽章。
  * **CNN**：五色連續漸層條，顯示昨收、上週與上月分數對比。
  * **AAII**：紅/灰/綠比例分割條，並標示 45% 看跌悲觀警戒線。
  * **NAAIM**：持倉位置指示線，低於 55/30 時自動亮起低點徽章。
* **高質感配色調整**：
  * 背景改為墨黑到星空灰藍的徑向漸層，增加空間感。
  * 卡片改為半透明玻璃擬態樣式，並加入平滑的 Hover 懸浮升起與陰影動畫。
  * 微調指示燈發光陰影，適配新的調色盤。

### ⚙️ 調整與修正
* **專案重命名**：將所有檔案（網頁、Actions、腳本、README）中的 `BOS` 全部替換為 `USA`。
* **歷史數據支援**：更新 `fetch_data.py` 讓 CNN 抓取腳本支援輸出上週、上月等歷史數值，供網頁端動態比對。

---

## [v1.0.0] - 2026-06-22

### 🎉 初始版本發布
* **核心資料抓取**：實作 `fetch_data.py` 每日自動抓取 VIX、CNN Fear & Greed、AAII、NAAIM 四大情緒指標，並計算綜合警示分數。
* **自動化排程**：建立 `.github/workflows/update.yml`，於美股開盤前自動執行腳本。
* **前端儀表板**：實作 `index.html` 讀取 `data.json` 數據並以三色警示燈、分數環呈現市場情緒。
